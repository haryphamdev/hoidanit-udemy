Câu hỏi thường gặp

  • Q1: Chưa biết gì về React/Node.JS, chỉ biết Javascript cơ bản, HTML, CSS nên học như thế nào ???
  • Học cái gì cũng vậy. Dục tốc bất đạt, đặc biệt với ngành CNTT. Mình hay bảo các bạn là, ngành CNTT nó không khó. Tuy nhiên, kiến thức nó nhiều, nên cần thời gian để tích lũy.
    Mình cũng đã trải qua cái thời của các bạn rồi. Tức là chẳng biết gì cả, nên mình hiểu điều các bạn đang trải nghiệm.
    Cơ mà, bây giờ về với thực tại, muốn biết, thì phải code. Lúc đầu, do chưa biết gì, nên thấy khó hiểu, thì chịu khó code theo.
    Trong giai đoạn này, mục đích đấy chính là mình thấy được những kiến thức nào cần học, cũng như thấy được thành quả sẽ đạt được.
    Làm 1 việc gì đấy, mà không thấy đích đến thì làm sao có động lực để làm, có phải không ?
    Nên là, cho dù các bạn thấy khó hiểu, thì cứ code theo đi. Để bản thân các bạn hình dung được con đường mình sẽ đi.
    Sau đấy, khi đã code theo rồi, mình đã hiểu sơ sơ rồi, thì bắt đầu tự làm project cá nhân.
    Cái nào quên thì cứ google. Ở đây, điểm khác biệt đấy chính là do các bạn đã code theo, có thể các bạn chưa code được như mình, cơ mà các bạn nhớ đươc keyword nhờ kiến thức đã học, nên có từ khóa mà search google.
    Vừa làm vừa search, làm lấy 1, 2 cái project cá nhân, thì kiến thức sẽ chuyển hóa thành là của mình. Đây chính là giai đoạn các bạn thấy dễ đấy ~.~
    Bí mật của code giỏi (biết code) là code nhiều thì quen tay, chứ chẳng có bí mật gì cả :v
